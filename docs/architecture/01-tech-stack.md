# Tech Stack

## App

- Expo
- React Native
- TypeScript
- Expo Router

## Backend

- Supabase Auth
- Supabase Postgres
- Supabase Storage
- Supabase Realtime

## Client State

- TanStack Query for server state
- Zustand for lightweight local UI state
- AsyncStorage for persisted lightweight client state

## Forms and Validation

- React Hook Form
- Zod

## Styling

- NativeWind for MVP styling speed
- Centralized design tokens
- Reanimated and Gesture Handler for motion and future timeline zoom

## Build and Deployment

- EAS Build
- EAS Submit

## Testing

- Jest
- React Native Testing Library
- Supabase local development for database/RLS testing

## Why This Stack

Daemi needs native mobile behavior, media picking, media upload, auth, realtime sharing, and private couple-scoped data. Expo and Supabase provide the fastest path to a production-capable MVP without requiring a custom backend.
